# 任务输入模板

默认只需要填写 `目标`。  
`验收标准 / 约束 / 非目标` 由执行方自动补齐并在开始节点声明。

## 目标

一句话描述你要达成的结果。

## 可选补充（非必填）

如果你希望覆盖自动默认值，可额外补充：

- 验收标准
- 约束
- 非目标

---

示例：

```text
目标：修复 AI 页面偶发黑屏，并保持左侧栏当前布局。
```

可选示例（覆盖默认值）：

```text
目标：修复 AI 页面偶发黑屏，并保持左侧栏当前布局。
验收标准：除了 verify:gate，通过后还要人工打开 ChatGPT/豆包各 1 次确认可见。
约束：本次允许增加 1 个调试依赖。
非目标：不调整侧栏视觉样式。
```
